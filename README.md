# Build a Word Counter

Practice using for...of loops by building a function that counts how often a string appears in an array of strings.

## Technologies

Uses JavaScript - `for...of loops`.

## Deployment

Copy `script.js` to the text editor of your choice.  If using terminal, use the command `node script.js` to run the code.
